Ejecución de la aplicación
=============

-  Clonar el repositorio:

```
    >git clone https://github.com/EduenSarceno/svelte-app.git
```

- Instalar dependencias

```
    >npm install
```

- Correr la aplicación localmente

```cmd
    >npm run dev -- --port 8080 --host 0.0.0.0 --open
```
